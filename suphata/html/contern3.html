อาหารและวัฒนธรรมอาหาร

อาหารเกาหลีหรือฮันซิก มีลักษณะเด่นคือมีการแบ่งแยกระหว่างอาหารจานหลัก (ข้าว เส้นก๋วยเตี๋ยว ฯลฯ) กับอาหารจานข้าง (เรียกว่าบันชัน ) อย่างชัดเจน มีสูตรอาหารที่หลากหลาย มีการผสมผสานผลิตภัณฑ์จากสัตว์และพืชอย่างสมดุล และอาหารหมักดองชาวเกาหลียังเชื่อกันว่าอาหารและยามีต้นกำเนิดเดียวกัน โดยยึดถือสุภาษิตที่ว่า "อาหารคือยาที่ดีที่สุด" อาหารหมักดองถือเป็นหัวใจสำคัญของการปรุงอาหารเกาหลี อาหารเหล่านี้ส่วนใหญ่เรียกว่า "ชัง" ซึ่งรวมถึงซอสถั่วเหลือง ( กันจัง ) เต้าเจี้ยว ( โดเอนจัง ) ซอสพริกแดง ( โคชูจัง ) ต้นหอม กระเทียม และพริกแดง ก็เป็นส่วนประกอบพื้นฐานของการปรุงอาหารเกาหลีเช่นกัน

1.กิมจิ

กิมจิเป็นหนึ่งในอาหารหลักของเกาหลีใต้ โดยปกติจะรับประทานเป็นเครื่องเคียง อาหารที่ประกอบด้วยข้าวสวยกับเครื่องเคียงหนึ่งถึงสามอย่าง (เช่น กิมจิ โดยเฉพาะอย่างยิ่ง แพชัน (กะหล่ำปลี)) เป็นเรื่องปกติ กิมจิมีรสชาติที่ผ่านการหมักเฉพาะตัว ไม่เพียงแต่เพราะสรรพคุณทางยาจากส่วนผสมเท่านั้น แต่ยังรวมถึงผลิตภัณฑ์หมักที่ให้รสชาติ เนื้อสัมผัส คุณภาพ และประโยชน์ต่อสุขภาพ อันเนื่องมาจากกระบวนการหมักแบบพิเศษ

2.ซุนดูบูจิเก

ซุนดูบูจิเกคือซุปเต้าหู้รสเผ็ดแบบเกาหลี แตกต่างจากซุปทั่วไป คือไม่ข้นหรือหนักเกินไป รสชาติอูมามิเข้มข้น ปรุงรสด้วยน้ำปลา กระเทียม ต้นหอม และเห็ด

3.บิบิมบับ

บิบิมบับเป็นอาหารที่ทำจากข้าว โดยมีผักและไข่หลากสีสันผสมอยู่ด้วย โดยเพิ่มโคชูจัง(พริกแดง) ลงไป ทำให้มีเนื้อสัมผัสที่แตกต่างไปจากเดิมมาก [ 34 ]แม้ว่าบิบิมบับมักจะเสิร์ฟพร้อมเนื้อสัตว์ แต่ชาวเกาหลีหลายคนก็รับประทานบิบิมบับโดยมีเพียงไข่เท่านั้นที่เป็นแหล่งโปรตีนในเมนูข้าวธรรมดา

4.ต็อกโบกี

ต็อกโบกี (ต็อกโบกีเกาหลี) เป็นอาหารริมทางยอดนิยมของเกาหลีใต้ ในอดีตต็อกโบกีถือเป็นอาหารชั้นเลิศที่ขุนนางและสมาชิกในราชสำนักรับประทาน แต่หลังจากสงครามเกาหลีในช่วงทศวรรษ 1950 ต็อกโบกีได้เปลี่ยนรูปแบบไปเป็นอาหารอีกประเภทหนึ่ง โดยทำโดยการต้มแป้งสาลีกับเค้กปลาและผักในซอสพริกแดง ต่อมาในช่วงทศวรรษ 1970 ต็อกโบกีได้กลายเป็นอาหารว่างของผู้คนจากทุกสาขาอาชีพ

5.บูแดจิเก

บูแดจิเกเป็นสตูว์ยอดนิยมที่มีต้นกำเนิดจากสงครามเกาหลี ส่วนประกอบหลักคล้ายกับสตูว์รสเผ็ดแบบดั้งเดิมของเกาหลีหลายชนิด เช่น ซุนดูบู หรือ กิมจิจิเก ซึ่งมักทำจากปลากะตัก สาหร่าย และโคชูจัง (พริกแดงเกาหลี) แช่รวมกัน แต่ส่วนผสมอื่นๆ ของบูแดจิเกก็ทำให้แตกต่างจากสตูว์ประเภทอื่นๆ สตูว์นี้ประกอบด้วยสแปม ถั่วอบ ฮอทดอก และเส้นราเมน โรยหน้าด้วยต้นหอม
